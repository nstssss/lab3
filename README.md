# Sleep Tracker - Приложение для отслеживания сна

Десктопное приложение для отслеживания и анализа качества сна с графическим интерфейсом, разработанное на Python с использованием PySide6.

## Описание

Sleep Tracker представляет собой полнофункциональное приложение для мониторинга и анализа параметров сна. Программа предоставляет удобный интерфейс для ведения дневника сна, отслеживания ключевых показателей и визуализации данных для анализа тенденций и закономерностей.

### Основные возможности

- Ведение записей о сне: добавление, редактирование и удаление записей
- Визуализация данных: построение графиков продолжительности и качества сна
- Статистический анализ: автоматический расчет средних показателей и общей статистики
- Локальное хранение данных: использование SQLite базы данных для надежного хранения информации
- Интуитивный интерфейс: простой и понятный графический интерфейс на основе PySide6

## Инструкции по запуску
-Скачайте или склонируйте файлы проекта
-Откройте терминал в папке с проектом
-Запустите главный файл приложения:
```bash
python sleep_tracker.py
```
### Предварительные требования

- Python 3.8 или выше
- pip (менеджер пакетов Python)

### Установка зависимостей
